# Change Password   

## Server 2019 AD/DNS - Server 2019 Web - Server 2022 FTP - Windows 11 USER 1  
Login with default creds, then send `ctrl + alt + delete` and click change password.   

## Windows 11 Admin
Open PowerShell in administrator mode   
`net user` to see all users 
`net user [username]*` and press enter.     
Afterwards you'll be able to change the password and confirm it.
